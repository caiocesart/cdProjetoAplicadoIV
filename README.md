## Descrição do projeto

Este projeto desenvolve um produto analítico de séries temporais para previsão de carga elétrica de consumidores, com base no dataset **Electricity Load Diagrams 2011-2014** (UCI). O objetivo é analisar padrões temporais (tendência, sazonalidade e variações de curto prazo) e apoiar decisões de planejamento energético, em alinhamento com os ODS 9 e 12.

## Setup

Este projeto utiliza o UV da Astral para gerenciamento de pacotes e dependências
https://docs.astral.sh/uv/getting-started/installation/

Utilização básica:
- `uv add pandas`
- `uv remove pandas`

Para garantir que seu ambiente está sincronizado com o repositório:
- `uv sync`
